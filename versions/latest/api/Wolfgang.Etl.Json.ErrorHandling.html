Controls how deserialization errors are handled during extraction.

public enum ErrorHandling

Fields

CaptureAndContinue = 1

Capture the error in Errors and continue processing remaining records where possible. For streaming formats (e.g. a JSON array), a parse error mid-stream may prevent continuation.

SkipAndLog = 2

Skip the error silently, logging a warning if a logger is configured.

Throw = 0

Throw on the first deserialization error (default).
